HOW TO LAUNCH PYTHON SCRIPT FOR CONVERT AND RESIZE JPG IMAGE TO PPM

Be careful, the script is using the OpenCV library, it’s a PyLib for image processing. 

Here is a link for the installation : https://pypi.org/project/opencv-python/

To launch the script, put your images and the script in the same directory. Then, go in terminal, and select the good path. 
	→ python3 conversion.py img.jpg img.ppm width height

Then, check your directory and the img.ppm will appear.
